# v0.4.5

## Fixes

- **Add missing language toolchain dotdirs to read-write list** — Several home dotdirs used by language toolchains were bind-mounted read-only, causing `bundle install`, `gem install`, `npm install`, etc. to fail with "File system is read-only". Added `.bundle`, `.gem`, `.rustup`, `.npm`, `.bun`, `.deno`, `.yarn`, `.pnpm`, `.m2`, `.gradle`, `.dotnet`, `.nuget`, `.pub-cache`, `.mix`, `.hex` to the read-write list.
